Decidido a disfrutar de este deporte, se le ocurrió la idea de construir una pista más pequeña rodeada de muros para mantener la pelota en juego.     ¿Qué es una red informática.?

Una red de computadoras, también llamada red de ordenadores o red informática, es un conjunto de equipos conectados por medio de cables, señales, ondas o cualquier otro método de transporte de datos, que comparten información (archivos), recursos.



    ¿Qué es una red LAN.?

Es una red contenida dentro de una pequeña zona geográfica, normalmente dentro del mismo edificio.


    ¿Qué es una red WAN?

Es la tecnología que conecta entre sí a las oficinas, los centros de datos, las aplicaciones en la nube y el almacenamiento en la nube.



    ¿Qué es es el Internet?

Red global de redes de ordenadores cuya finalidad es permitir el intercambio libre de información entre todos sus usuarios.


    ¿Qué es la web?

Es un sistema de información en línea que permite el acceso a una enorme cantidad de recursos y contenidos a través de internet.



    ¿Qué es WWW?

Es un sistema interconectado de páginas web públicas accesibles a través de Internet.



    ¿Qué es el protocolo HTTP y HTTPS.?

Los mensajes HTTP son de texto plano, lo que significa que las personas no autorizadas pueden acceder a ellos y leerlos fácilmente a través de Internet. Por el contrario, HTTPS transmite todos los datos de forma cifrada.



    ¿Página web estática vs página web dinámica.?

A diferencia de las páginas web estáticas, que muestran el mismo contenido para todos los usuarios, las páginas web dinámicas pueden personalizarse y cambiar en tiempo real.



    ¿CMS. Ejemplos. Ventajas e inconvenientes.?

Ahorrarle tiempo al equipo de desarrollo, y actualizar directamente el sitio. Es un sistema online que permite poner en marcha una página web de forma sencilla y rápida.



    ¿Proveedor de servicios de Internet. Ejemplos.?

Es la empresa que te brinda el acceso a Internet. Movistar, Vodafone, Orange, Yoigo, DIGI.



    ¿Dirección IP. Diferencia entre una dirección IP privada y pública.?

Una dirección IP pública lo identifica en Internet, de modo que toda la información que busque pueda llegar hasta usted. Una dirección IP privada se utiliza dentro de una red privada para conectarse de forma segura a otros dispositivos de la misma red.



    ¿Qué es el alojamiento web/hosting? Ejemplos.

Un hosting es un servicio de alojamiento web que te permite publicar un sitio web o aplicación en Internet. Cuando contratas un servicio de hosting, básicamente alquilas un espacio en un servidor físico donde puedes almacenar todos los archivos y datos necesarios para que tu sitio web funcione correctamente.
